# vivado_lab

vivado_lab installed into a docker image for CI purposes.

## Install

1. Clone

```
git clone --recursive https://github.com/xiongyumail/vivado_lab.git
```
2. Install docker.io

```
sudo apt install docker.io
```

## Running

```
./vivado_lab.sh
```

or

```
./vivado_lab.sh [projects_path]
```

## Clean

```
./vivado_lab.sh clean
```
